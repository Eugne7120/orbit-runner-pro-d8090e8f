# 0RBIT — Sprint 5
## Build the Worker Dashboard & Network Experience

Do NOT redesign the homepage.

Do NOT modify the design system.

Do NOT change typography.

Do NOT change colors.

Do NOT replace existing animations.

The goal of this sprint is to build a complete Worker Network experience that shows how the decentralized compute network operates.

The page should feel like a real infrastructure dashboard.

Everything should be frontend only.

No backend is required.
